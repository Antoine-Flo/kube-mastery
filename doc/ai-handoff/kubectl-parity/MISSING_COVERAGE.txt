# Liste simple des manques

Supprimer une ligne uniquement quand la parity stricte est validée sur kind.

RBAC
- kubectl auth reconcile -f <file>

Jobs
- kubectl create job <name> --image=busybox
- kubectl get jobs
- kubectl describe job <name>
- kubectl delete job <name>
- kubectl get cronjobs
- kubectl describe cronjob <name>
- kubectl create cronjob <name> --image=busybox --schedule="*/5 * * * *"
- kubectl delete cronjob <name>

Auth

Certificates
- kubectl certificate approve <csr>
- kubectl certificate deny <csr>
- kubectl get csr
- kubectl describe csr <name>

Helm
- helm version
- helm repo add <name> <url>
- helm repo update
- helm search repo
- helm install <release> <chart>
- helm list
- helm status <release>
- helm upgrade <release> <chart>
- helm uninstall <release>
- helm rollback <release> <revision>
- helm template <release> <chart>
- helm pull <chart>
- helm dependency update <chart-dir>

Port-forward
- kubectl port-forward pod/<name> 8080:80
- kubectl port-forward service/<name> 8080:80

Node operations
- kubectl cordon <node>
- kubectl uncordon <node>
- kubectl drain <node> --ignore-daemonsets --delete-emptydir-data
- kubectl taint nodes <node> key=value:NoSchedule
- kubectl taint nodes <node> key:NoSchedule-

Debug and troubleshooting
- kubectl cp <src> <pod>:<dst>
- kubectl cp <pod>:<src> <dst>
- kubectl attach <pod> -c <container> -it
- kubectl debug pod/<name> -it --image=busybox

Context and kubeconfig

Plugin and extension surface
- kubectl plugin list
- kubectl kustomize <dir>

Admission and policy related resources
- kubectl get poddisruptionbudgets
- kubectl describe poddisruptionbudget <name>
- kubectl get priorityclasses
- kubectl describe priorityclass <name>

Custom resources and CRD
- kubectl get crd
- kubectl describe crd <name>
- kubectl get <crd-plural>

Authentication plumbing and identity

Autoscaling
- kubectl autoscale deployment <name> --cpu-percent=50 --min=1 --max=10
- kubectl get hpa
- kubectl get horizontalpodautoscalers
- kubectl describe hpa <name>

Resource limits and quotas
- kubectl get limitranges
- kubectl describe limitrange <name>
- kubectl get resourcequotas
- kubectl describe resourcequota <name>

Rollout completeness
- kubectl rollout pause deployment/<name>
- kubectl rollout resume deployment/<name>

kubectl set family
- kubectl set env deployment/<name> KEY=VALUE
- kubectl set env deployment/<name> KEY-
- kubectl set resources deployment/<name> --limits=cpu=200m,memory=512Mi --requests=cpu=100m,memory=256Mi
- kubectl set serviceaccount deployment/<name> <sa-name>

Patch type variants
- kubectl patch <resource>/<name> --type=json -p '[{"op":"replace","path":"/spec/replicas","value":3}]'
- kubectl patch <resource>/<name> --type=strategic -p '<json>'
