# Plan du Module Policy

## Chapitres

### 01-resource-quotas

Quotas de ressources : ResourceQuota par namespace, comptage d'objets, quotas de compute, scopes.

### 02-limit-ranges

Limites par défaut : LimitRange, defaults et defaultRequests, min/max, ratio limite/request.

### 03-pid-limiting

Limitation des PIDs : protection contre les fork bombs, configuration kubelet, réservation de PIDs.

### 04-node-resource-managers

Gestion des ressources node : CPU Manager, Memory Manager, Device Manager, Topology Manager.
