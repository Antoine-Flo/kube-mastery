# Plan du Module Windows

## Chapitres

### 01-introduction
Windows dans Kubernetes : support, limitations, compatibilité des versions Windows Server.

### 02-user-guide
Guide utilisateur : scheduling des containers Windows, node selectors, taints/tolerations pour Windows.

### 03-networking
Réseau Windows : particularités, CNI plugins supportés, différences avec Linux networking.

### 04-storage
Stockage Windows : volumes supportés, CSI drivers Windows, particularités.

### 05-security
Sécurité Windows : RunAsUserName, Group Managed Service Accounts (gMSA), HostProcess containers.

### 06-resource-management
Gestion des ressources : CPU et mémoire sur Windows, différences avec Linux, limitations.

### 07-debugging
Debugging Windows : troubleshooting, logs, outils de diagnostic spécifiques Windows.
