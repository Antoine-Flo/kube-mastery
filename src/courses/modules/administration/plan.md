# Plan du Module Administration

## Chapitres

### 01-addons
Extensions du cluster : DNS, Dashboard, monitoring, logging addons, installation et gestion.

### 02-certificates
Gestion des certificats : PKI Kubernetes, génération, rotation, Certificate Signing Requests.

### 03-logging
Logging cluster : architecture de logging, logging au niveau node et cluster, solutions (EFK, Loki).

### 04-cluster-networking
Réseau cluster : modèle réseau Kubernetes, CNI plugins, configuration, troubleshooting.

### 05-node-autoscaling
Scaling automatique des nodes : Cluster Autoscaler, scaling up/down, configuration, cloud providers.

### 06-node-shutdown
Arrêt des nodes : graceful node shutdown, non-graceful shutdown handling, draining.

### 07-observability
Observabilité : metrics, tracing, health endpoints, monitoring du control plane.

### 08-system-metrics
Métriques système : Metrics API, metrics-server, resource metrics, custom metrics.

### 09-system-logs
Logs système : composants du control plane, format des logs, configuration, analyse.

### 10-flow-control
Contrôle du flux API : API Priority and Fairness, PriorityLevelConfigurations, FlowSchemas.

### 11-admission-webhooks
Webhooks d'admission : bonnes pratiques, performance, fiabilité, debugging.
