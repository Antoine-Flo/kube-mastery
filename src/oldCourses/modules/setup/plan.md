# Plan du Module Setup

## Chapitres

### 01-learning-environment

Environnements d'apprentissage : minikube, kind, k3s, Docker Desktop, choix selon les besoins.

### 02-production-environment

Environnements de production : considérations (HA, sécurité, networking), choix d'architecture.

### 03-kubeadm

Installation avec kubeadm : initialisation du cluster, ajout de nodes, configuration, HA setup.

### 04-container-runtimes

Configuration des runtimes : containerd, CRI-O, installation et configuration.

### 05-turnkey-solutions

Solutions managées : EKS, GKE, AKS, autres providers, comparaison, migration.

### 06-best-practices

Bonnes pratiques : sizing, networking, sécurité, monitoring, backup/restore.
